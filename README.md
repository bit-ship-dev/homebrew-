# homebrew-bit-ship

Homebrew tap for [bit-ship](https://www.bit-ship.dev) — a containerized deployment tool.

## Install

```sh
brew tap bit-ship-dev/ && brew install bit-ship
```

## Requirements

- [Node.js](https://nodejs.org) — installed automatically via Homebrew
- [Podman](https://podman.io) — installed automatically via Homebrew

## What it does

`brew install bit-ship` runs `npm install -g bit-ship` under the hood, placing the binary in your Homebrew prefix.
